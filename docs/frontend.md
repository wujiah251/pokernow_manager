# 前端功能文档

## 页面结构

单页应用（SPA），共5个标签页：

### 1. 玩家映射管理
- 查看所有玩家的 nickname 和 alias 映射关系
- 添加新的 nickname ← alias 映射
- 点击表头可排序

### 2. PnL 查询
- 选择日期查看当日所有玩家的净盈亏
- 可按玩家筛选
- 表格显示：日期、玩家、净盈亏
- 点击表头可排序（默认按净盈亏降序）

### 3. 累计盈亏曲线
- 选择日期范围
- 可选择特定玩家（可选）
- 选择玩家：显示该玩家的累计盈亏曲线
- 未选择玩家：显示所有玩家的累计盈亏曲线（ECharts 多线图）
- 支持点击图例显示/隐藏特定玩家的曲线

### 4. 数据上传
- 选择日期
- 选择 CSV 文件上传
- 支持 ledger 格式文件上传
- 上传后自动计算每日 PnL

### 5. 数据删除
- 选择日期范围
- 可选择删除 PnL 记录和/或 Ledger 记录
- 执行删除操作

## 技术栈

- Vue 3 (本地化)
- ECharts (图表)
- 原生 JavaScript
